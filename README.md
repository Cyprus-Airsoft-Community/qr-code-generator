# Generated QR Code Invitation for our Discord Server

## Step 1

    Create the virtual environment:

python -m venv venv

## Step 2

    Activate it:

venv\Scripts\activate

## Step 3

    Install the libraries used:

python -m pip install -r requirements.txt

## Step 4

    Run the script:

python cac-qr-code.py
